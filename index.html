Responsive Locator Map Generator v0.1

Quick responsive locator map.

Enter an complete address or a pair of coordinates coordinates (comma delimited, include decimals). Start in Boston or see examples.

Methode best practices

  • Paste in the above code
  • Insert jpt into story in desired location
  • Talk to a producer on how to slot in a jpt
